## Linux研修【サーバ構築ハンズオン】Web編(nginx/apache/Squid)
- CentOS8をインスト―ルしてネットワーク接続するまで
    - インストールの後
        - タイムゾーンの設定を確認する
        ```
        # timedatectl status
        ```
        - タイムゾーンを日本に変更する
        ```
        # timedatectl set-timezone Asia/Tokyo
        ```
        - デバイスの状態を確認する
        ```
        # nmcli d
        ```
